# dskripchenko/laravel-admin-starter

> 🌐 [English](README.md) · [Русский](README.ru.md) · **Deutsch** · [中文](README.zh.md)

Sofort einsatzbereite Admin-Resourcen: Users, Roles, AuditLog, Settings, Translations, ContentBlocks. Plug-and-Play-Startpunkt für neue Admins.

Ein Sister-Pack für [`dskripchenko/laravel-admin`](https://github.com/dskripchenko/laravel-admin).

## Installation

```bash
composer require dskripchenko/laravel-admin-starter
php artisan migrate
```

Das Plugin registriert sich automatisch über Laravel Package Discovery.
